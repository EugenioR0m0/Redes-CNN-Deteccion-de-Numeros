Preprocesamiento del dataset

Se definen las rutas de los conjuntos original y preprocesado. Se crean las carpetas de salida y se implementa una funcin para preprocesar cada imagen del dataset. El preprocesamiento consiste en:

  • Leer la imagen en escala de grises.

  • Redimensionarla a 280280.

  • Aplicar binarizacin con Otsu.

  • Invertir los colores para mantener el formato esperado por el modelo.

  • Guardar cada imagen procesada en su carpeta correspondiente.

Finalmente, se ejecuta la funcin para procesar tanto el conjunto de entrenamiento como el de prueba.

def preprocess_and_save(src_folder, dst_folder):
    for digit in os.listdir(src_folder):
        src_digit_path = os.path.join(src_folder, digit)
        dst_digit_path = os.path.join(dst_folder, digit)
        os.makedirs(dst_digit_path, exist_ok=True)

        for img_name in tqdm(os.listdir(src_digit_path), desc=f"Procesando {digit}"):
            img_path = os.path.join(src_digit_path, img_name)

            img = cv2.imread(img_path, cv2.IMREAD_GRAYSCALE)

            img = cv2.resize(img, (280, 280), interpolation=cv2.INTER_AREA)
            _, img = cv2.threshold(img, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
            img = cv2.bitwise_not(img)

            cv2.imwrite(os.path.join(dst_digit_path, img_name), img)

# RUN
preprocess_and_save(original_train, output_train)
preprocess_and_save(original_test,  output_test)

Modelo Baseline

Se define un modelo CNN bsico compuesto por tres bloques convolucionales con activacin ReLU y max pooling. Despus de la extraccin de caractersticas, la red se aplana y pasa por una capa densa intermedia con dropout para reducir sobreajuste. Finalmente, una capa softmax produce la clasificacin en 10 clases.

El modelo se entrena durante 60 pocas utilizando los generadores de entrenamiento y validacin, con EarlyStopping para detener el proceso si no hay mejora. Al finalizar, el modelo y su historial de entrenamiento se guardan para anlisis posterior.

def build_baseline_model(input_shape=(280,280,1), num_classes=10):

    inputs = keras.Input(shape=input_shape)

    # Bloque 1
    x = layers.Conv2D(32, (3,3), padding='same')(inputs)
    x = layers.ReLU()(x)
    x = layers.MaxPooling2D(2,2)(x)

    # Bloque 2
    x = layers.Conv2D(64, (3,3), padding='same')(x)
    x = layers.ReLU()(x)
    x = layers.MaxPooling2D(2,2)(x)

    # Bloque 3
    x = layers.Conv2D(128, (3,3), padding='same')(x)
    x = layers.ReLU()(x)
    x = layers.MaxPooling2D(2,2)(x)

    # Clasificacin
    x = layers.Flatten()(x)
    x = layers.Dense(128, activation='relu')(x)
    x = layers.Dropout(0.3)(x)

    outputs = layers.Dense(num_classes, activation='softmax')(x)

    model = keras.Model(inputs, outputs)

    model.compile(
        optimizer='adam',
        loss='categorical_crossentropy',
        metrics=['accuracy']
    )
    return model

Modelo VGG Mejorado

Se implementa una versin optimizada del modelo tipo VGG, incorporando inicializacin He Normal, capas de normalizacin por lotes y dropout progresivo para estabilizar el entrenamiento y reducir sobreajuste.

La arquitectura consta de cuatro bloques convolucionales con dos capas por bloque, seguidos de GlobalAveragePooling para reducir dimensionalidad sin introducir demasiados parmetros. La etapa final incluye una capa densa con activacin relu y una capa de salida softmax para clasificar en 10 clases.

El entrenamiento utiliza Adam con un learning rate reducido para mejorar la estabilidad, junto con EarlyStopping para mantener los mejores pesos. El historial del entrenamiento se guarda para su anlisis posterior y el modelo se exporta en formato .keras.

def build_vgg_model(input_shape=(280,280,1), num_classes=10):
    inputs = keras.Input(shape=input_shape)

    # Bloque 1
    x = layers.Conv2D(32, (3,3), padding='same', kernel_initializer='he_normal')(inputs)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)
    x = layers.Conv2D(32, (3,3), padding='same', kernel_initializer='he_normal')(x)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)
    x = layers.MaxPooling2D(2,2)(x)
    x = layers.Dropout(0.1)(x)

    # Bloque 2
    x = layers.Conv2D(64, (3,3), padding='same', kernel_initializer='he_normal')(x)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)
    x = layers.Conv2D(64, (3,3), padding='same', kernel_initializer='he_normal')(x)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)
    x = layers.MaxPooling2D(2,2)(x)
    x = layers.Dropout(0.15)(x)

    # Bloque 3
    x = layers.Conv2D(128, (3,3), padding='same', kernel_initializer='he_normal')(x)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)
    x = layers.Conv2D(128, (3,3), padding='same', kernel_initializer='he_normal')(x)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)
    x = layers.MaxPooling2D(2,2)(x)
    x = layers.Dropout(0.20)(x)

    # Bloque 4
    x = layers.Conv2D(256, (3,3), padding='same', kernel_initializer='he_normal')(x)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)
    x = layers.MaxPooling2D(2,2)(x)
    x = layers.Dropout(0.25)(x)

    # Clasificacin
    x = layers.GlobalAveragePooling2D()(x)
    x = layers.Dense(256, activation='relu', kernel_initializer='he_normal')(x)
    x = layers.Dropout(0.45)(x)

    outputs = layers.Dense(num_classes, activation='softmax')(x)

    model = keras.Model(inputs, outputs)

    optimizer = keras.optimizers.Adam(learning_rate=1e-3)

    model.compile(
        optimizer=optimizer,
        loss='categorical_crossentropy',
        metrics=['accuracy']
    )
    return model

Modelo ResNet Simplificado

Se implementa un modelo basado en bloques residuales para mejorar el flujo del gradiente durante el entrenamiento y reducir problemas de desvanecimiento. Cada bloque residual aplica dos convoluciones con normalizacin por lotes, combinadas mediante conexiones tipo skip que permiten preservar informacin de capas anteriores.

El modelo inicia con una convolucin inicial para extraccin temprana de caractersticas, seguida de dos bloques residuales con incrementos progresivos en filtros. Posteriormente, se emplea GlobalAveragePooling para reducir dimensionalidad sin aadir demasiados parmetros. Finalmente, una capa densa con dropout prepara las caractersticas para la clasificacin en 10 clases mediante softmax.

El entrenamiento utiliza el optimizador Adam, prdida categorical crossentropy y la mtrica accuracy. Se aplica EarlyStopping para retener los mejores pesos segn el desempeo en validacin, y se guarda el historial en un diccionario para anlisis posterior. El modelo entrenado se almacena en formato .keras.

def residual_block(x, filters):
    skip = x

    x = layers.Conv2D(filters, (3,3), padding='same', activation='relu')(x)
    x = layers.BatchNormalization()(x)
    x = layers.Conv2D(filters, (3,3), padding='same')(x)
    x = layers.BatchNormalization()(x)

    # Proyeccin del skip para que coincidan dimensiones
    skip = layers.Conv2D(filters, (1,1), padding='same')(skip)

    out = layers.Add()([x, skip])
    out = layers.ReLU()(out)
    return out


def build_resnet_model(input_shape=(280,280,1), num_classes=10):
    inputs = keras.Input(shape=input_shape)

    x = layers.Conv2D(32, (3,3), strides=1, padding='same')(inputs)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)

    x = residual_block(x, 32)
    x = layers.MaxPooling2D(2,2)(x)

    x = residual_block(x, 64)
    x = layers.MaxPooling2D(2,2)(x)

    x = residual_block(x, 128)
    x = layers.GlobalAveragePooling2D()(x)

    x = layers.Dense(128, activation='relu')(x)
    x = layers.Dropout(0.3)(x)

    outputs = layers.Dense(num_classes, activation='softmax')(x)
    model = keras.Model(inputs, outputs)

    model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])
    return model

Modelo Inception Simplificado

Este modelo implementa un bloque tipo Inception, donde varios caminos paralelos extraen caractersticas con distintos tamaos de filtros. Cada bloque combina convoluciones de 11, 33, 55 y un camino de max pooling seguido de convolucin 11. Esta arquitectura permite capturar informacin espacial en mltiples escalas sin aumentar excesivamente el costo computacional.

El modelo inicia con un bloque Inception de baja dimensionalidad para la extraccin de rasgos bsicos, seguido de bloques con mayor nmero de filtros para aprender representaciones ms complejas. Despus de la etapa convolucional, se usa GlobalAveragePooling para reducir dimensionalidad preservando las caractersticas relevantes.

Finalmente, se incorpora una capa densa con activacin relu y dropout, antes de la capa de salida con activacin softmax para clasificar en 10 clases.

def inception_block(x, f1, f3, f5):
    path1 = layers.Conv2D(f1, (1,1), padding='same', activation='relu')(x)

    path3 = layers.Conv2D(f3, (1,1), padding='same', activation='relu')(x)
    path3 = layers.Conv2D(f3, (3,3), padding='same', activation='relu')(path3)

    path5 = layers.Conv2D(f5, (1,1), padding='same', activation='relu')(x)
    path5 = layers.Conv2D(f5, (5,5), padding='same', activation='relu')(path5)

    pool = layers.MaxPooling2D((3,3), strides=1, padding='same')(x)
    pool = layers.Conv2D(f1, (1,1), padding='same', activation='relu')(pool)

    return layers.concatenate([path1, path3, path5, pool])


def build_inception_model(input_shape=(280,280,1), num_classes=10):
    inputs = keras.Input(shape=input_shape)

    x = inception_block(inputs, 32, 32, 32)
    x = layers.MaxPooling2D(2,2)(x)

    x = inception_block(x, 64, 64, 64)
    x = layers.MaxPooling2D(2,2)(x)

    outputs = layers.Dense(num_classes, activation='softmax')(x)
    model = keras.Model(inputs, outputs)

    model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])
    return model

Modelo MobileNet Simplificado

Este modelo implementa una versin reducida del enfoque de MobileNet, basada en depthwise separable convolutions. Este tipo de convoluciones divide la operacin estndar en dos etapas:

  1. Depthwise Convolution: aplica un filtro por canal, reduciendo drsticamente el costo computacional.
  2. Pointwise Convolution (11): combina las caractersticas generadas en el paso anterior.

Este diseo permite construir redes eficientes y livianas sin sacrificar demasiada precisin, lo cual es adecuado para tareas de reconocimiento de dgitos en imgenes sencillas.

El modelo inicia con una convolucin estndar para ampliar canales, seguida de una secuencia de bloques depthwise con incrementos progresivos en filtros y algunas capas con stride=2 para reducir la resolucin espacial. Despus de la etapa convolucional, se aplica GlobalAveragePooling para compactar las caractersticas aprendidas y una capa densa con activacin relu, complementada con dropout para reducir sobreajuste.

La salida se realiza mediante una capa softmax con 10 neuronas correspondientes a las clases del dataset. El entrenamiento usa Adam, prdida categorical crossentropy y accuracy como mtrica. Se emplea EarlyStopping para detener el entrenamiento cuando no hay mejora y se guardan los mejores pesos. Finalmente, el modelo entrenado se exporta en formato .keras y su historial se almacena para anlisis posterior.

def depthwise_block(x, filters, stride=1):
    x = layers.DepthwiseConv2D((3,3), strides=stride, padding="same")(x)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)

    x = layers.Conv2D(filters, (1,1), padding="same")(x)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)
    return x


def build_mobilenet_model(input_shape=(280,280,1), num_classes=10):
    inputs = keras.Input(shape=input_shape)

    x = layers.Conv2D(16, (3,3), strides=2, padding="same")(inputs)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)

    x = depthwise_block(x, 32)
    x = depthwise_block(x, 64, stride=2)
    x = depthwise_block(x, 128, stride=2)
    x = depthwise_block(x, 128)
    x = depthwise_block(x, 256, stride=2)

    x = layers.GlobalAveragePooling2D()(x)
    x = layers.Dense(128, activation='relu')(x)
    x = layers.Dropout(0.3)(x)

    outputs = layers.Dense(num_classes, activation='softmax')(x)
    model = keras.Model(inputs, outputs)

    model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])
    return model

Modelo CNN con Aumento de Datos

Este modelo implementa una arquitectura convolucional estndar reforzada con aumento de datos para mejorar la capacidad de generalizacin. El preprocesamiento incluye operaciones como volteo horizontal, rotacin ligera y zoom aleatorio, aplicadas en tiempo real antes de cada pasada por la red. Estas transformaciones permiten que el modelo sea ms robusto ante variaciones de orientacin, escala y pequeas deformaciones en los dgitos.

La arquitectura consta de tres bloques convolucionales. Cada bloque incluye dos capas Conv2D sin bias (compensado mediante Batch Normalization) seguidas de funciones de activacin ReLU y una capa MaxPooling para reduccin espacial. En cada bloque se incrementa progresivamente el nmero de filtros para capturar patrones de mayor complejidad.

Luego de la etapa convolucional, se aplica GlobalAveragePooling como mtodo eficiente de reduccin, seguido de una capa densa intermedia con activacin relu y dropout para mitigar sobreajuste. Finalmente, la capa softmax produce las probabilidades para las 10 clases.

El modelo se entrena con Adam utilizando una tasa de aprendizaje reducida (1e-4), emplea categorical crossentropy como funcin de prdida y accuracy como mtrica de desempeo. Se utiliza EarlyStopping para detener el entrenamiento cuando ya no existe mejora, y los mejores pesos son guardados en formato .keras. El historial de entrenamiento queda almacenado para anlisis posterior.

def build_augmented_cnn_model(input_shape=(280,280,1), num_classes=10):

    data_augmentation = keras.Sequential([
        layers.RandomFlip("horizontal"),
        layers.RandomRotation(0.05),
        layers.RandomZoom(0.1)
    ])

    inputs = keras.Input(shape=input_shape)
    x = data_augmentation(inputs)

    # Bloque 1
    x = layers.Conv2D(32, (3,3), padding='same', use_bias=False)(x)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)
    x = layers.Conv2D(32, (3,3), padding='same', use_bias=False)(x)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)
    x = layers.MaxPooling2D((2,2))(x)

    # Bloque 2
    x = layers.Conv2D(64, (3,3), padding='same', use_bias=False)(x)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)
    x = layers.Conv2D(64, (3,3), padding='same', use_bias=False)(x)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)
    x = layers.MaxPooling2D((2,2))(x)

    # Bloque 3
    x = layers.Conv2D(128, (3,3), padding='same', use_bias=False)(x)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)

    x = layers.ReLU()(x)

    x = layers.Conv2D(256, (3,3), padding='same', kernel_initializer='he_normal')(x)
    x = layers.BatchNormalization()(x)
    x = layers.ReLU()(x)
    x = layers.MaxPooling2D(2,2)(x)
    x = layers.Dropout(0.25)(x)

    # Clasificacin
    x = layers.GlobalAveragePooling2D()(x)
    x = layers.Dense(256, activation='relu', kernel_initializer='he_normal')(x)
    x = layers.Dropout(0.45)(x)

    outputs = layers.Dense(num_classes, activation='softmax')(x)

    model = keras.Model(inputs, outputs)

    optimizer = keras.optimizers.Adam(learning_rate=1e-3)

    model.compile(
        optimizer=optimizer,
        loss='categorical_crossentropy',
        metrics=['accuracy']
    )
    return model

model_final = build_vgg_model()

    callbacks=[early_stop_final]
